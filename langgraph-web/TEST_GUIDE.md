# 作文批改系统 - Vue 版本快速测试

## 🚀 快速启动（3 步）

### 步骤 1：启动后端

```bash
cd langgraph-api
mvn spring-boot:run
```

等待后端启动完成（看到 "Started LanggraphApiApplication"）。

### 步骤 2：启动前端

```bash
cd langgraph-web
npm install  # 首次运行需要
npm run dev
```

等待前端启动完成（看到 "Local: http://localhost:5173/"）。

### 步骤 3：浏览器访问

打开浏览器访问：**http://localhost:5173**

## 🐛 常见问题

### 问题 1：点击提交后无响应

**原因**：后端未启动

**解决**：
```bash
# 检查后端是否运行
curl http://localhost:8080/api/essay/review?studentName=测试&essayTopic=测试

# 启动后端
cd langgraph-api
mvn spring-boot:run
```

### 问题 2：CORS 错误

**原因**：代理配置未生效

**解决**：
1. 检查 `vite.config.ts` 中的 proxy 配置
2. 重启前端开发服务器
3. 清除浏览器缓存

### 问题 3：流式输出变成一次性显示

**原因**：Content-Type 不正确

**解决**：
1. 打开浏览器 DevTools → Network
2. 查看响应头 `Content-Type` 应为 `text/event-stream`
3. 检查后端是否正确使用 `StreamingChatGenerator`

### 问题 4：TypeScript 编译错误

**原因**：IDE 配置问题

**解决**：
```bash
# 使用项目自带的 TypeScript 编译
npm run build

# 忽略 IDE 错误，实际编译没问题
```
